# Katcha Chill and Fitness App

![Main](./previews/Mains.png)

## Description
Katcha Chill Fitness App is a mobile-first gym management application developed for a local gym client as part of a university project.

The system was designed to modernize gym operations by replacing manual attendance tracking and package management with a realtime mobile platform featuring QR-based check-ins, credit packages, store, and customer engagement systems.
